# table_epi_comparison_working note

This is a working comparison table for 03_LOGO_EPI.

Paper side:
- The main paper states that LOGO-EPI improves over DeepTACT by 0.23–4.47% absolute AUPRC across six cell lines.
- The largest improvement is reported for tCD4.
- Exact per-cell AUPRC values are referenced in Supplementary Table S10.

Reproduced side:
- Standard evaluation is treated as the main reproduced result.
- Balanced evaluation is treated as supplementary analysis, because it uses cell-type-specific threshold optimization.
- FoeT and Mon standard metrics are currently missing from the recovered logs.
